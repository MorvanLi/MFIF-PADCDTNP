MFFW (multi-focus image fusion in the wild) is a dataset for multi-focus image fusion to test whether an algorithm could deal with defocus spread effect. Please kindly cite the following paper if you use MFFW.

[1] Shuang Xu, Lizhen Ji, Zhe Wang, Pengfei Li, Kai Sun, Chunxia Zhang, Jiangshe Zhang:
Towards Reducing Severe Defocus Spread Effects for Multi-Focus Image Fusion via an Optimization Based Strategy. IEEE Trans. Computational Imaging 6: 1561-1570 (2020)
[2] Shuang Xu, Xiaoli Wei, Chunxia Zhang, Junmin Liu, Jiangshe Zhang:
MFFW: A new dataset for multi-focus image fusion. arxiv: 2002.04780 (2020)
